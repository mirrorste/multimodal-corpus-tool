# 多模态语料获取工具 - 需求设计文档

## 1. 项目概述

### 1.1 项目背景
本工具面向视听翻译、多模态隐喻与不可译性领域的研究者，提供从视频资源到标准化多模态语料的完整解决方案，支持质性分析与量化统计研究。

### 1.2 项目目标
构建一个自动化工具链，实现从视频获取到多模态语料输出的全流程处理，为研究者提供高质量、标准化的研究数据源。

### 1.3 目标用户
- 视听翻译研究者
- 多模态隐喻分析学者
- 不可译性研究人员
- 计算语言学研究者

| 路径 | 方法 | 说明 |

| `/api/videos` | POST | 创建视频下载任务 |

## 附录

### A. 术语定义

| 术语 | 定义 |
| :--- | :--- |
| 多模态 | 涉及两种或多种模态（文本、图像、音频等）的信息表示 |
| 隐喻 | 用一个概念域（源域）来理解和表达另一个概念域（目标域） |
| 不可译性 | 由于语言、文化或语境差异导致的翻译困难或无法翻译 |
| OCR | 光学字符识别，从图像中提取文本 |
| ASR | 自动语音识别，从音频中提取文本 |
| 语料 | 用于语言研究的大量语言材料 |
| 对齐 | 将不同模态的内容按照时间或语义关联起来 |

- ISO 12620:2019 术语数据交换格式
- TEI P5 文本编码倡议
