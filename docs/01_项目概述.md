**版本**: v0.1.0
**更新日期**: 2026-06-30
**状态**: 开发中

---

## 一、项目简介

### 1.1 项目背景

本工具面向视听翻译、多模态隐喻与不可译性领域的研究者，提供从视频资源到标准化多模态语料的完整解决方案。系统支持视频下载、帧提取、字幕识别、语音转写、视觉分析、多模态对齐、隐喻识别和不可译性分析等功能，为质性分析与量化统计研究提供高质量的语料数据。

### 1.2 项目目标

构建一个模块化、高扩展性的自动化工具链，实现从视频获取到多模态语料输出的全流程处理。系统采用微服务架构，支持 Docker 容器化部署，可本地运行或云端部署。

### 1.3 目标用户

- 视听翻译研究者
- 多模态隐喻分析学者
- 不可译性研究人员
- 计算语言学研究者
- 自然语言处理工程师

## 二、技术架构

### 2.1 技术栈

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| **后端框架** | FastAPI 0.115+ | 高性能异步 API 框架 |
| **数据库** | PostgreSQL 15+ | 关系型数据库 |
| **缓存/队列** | Redis 7+ / Celery | 异步任务队列 |
| **对象存储** | MinIO | S3 兼容对象存储 |
| **前端框架** | React 18 + TypeScript | 组件化开发 |
| **UI 组件** | Ant Design 5 | 企业级 UI 组件库 |
| **视频下载** | yt-dlp | 多平台视频下载 |
| **视频处理** | FFmpeg | 音视频处理 |
| **OCR** | PaddleOCR | 多语言文字识别 |
| **ASR** | Whisper | 语音识别 |
| **视觉模型** | BLIP-2, YOLOv8 | 图像描述、物体检测 |
| **NLP** | Transformers | 隐喻识别、语义分析 |

## 三、功能模块

### 3.1 视频资源获取模块

支持从多个视频平台下载视频资源：

| 功能 | 说明 | 优先级 |
|------|------|--------|
| YouTube 下载 | yt-dlp 集成，支持多种分辨率 | 高 |
| Bilibili 下载 | 支持番剧、弹幕下载 | 高 |
| Vimeo 下载 | 支持专业视频下载 | 中 |
| 本地文件导入 | 支持 MP4/AVI/MKV 等格式 | 高 |
| 批量下载 | 支持批量导入 URL 列表 | 高 |
| 断点续传 | 网络中断后自动恢复 | 低 |

### 3.2 帧提取模块

从视频中提取关键帧用于后续分析：

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 帧提取 | FFmpeg 驱动，支持自定义 FPS | 高 |
| 质量检测 | 自动过滤模糊帧 | 高 |
| 字幕帧检测 | 识别包含字幕的画面 | 高 |
| 关键帧提取 | 场景变化检测 | 中 |

### 3.3 多模态内容提取模块

#### 3.3.1 字幕提取

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 内嵌字幕 | FFmpeg 提取字幕轨道 | 高 |
| OCR 识别 | PaddleOCR 识别硬字幕 | 高 |
| 多语言支持 | 中英日韩等语言 | 高 |

#### 3.3.2 音频提取

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 音频提取 | WAV/MP3 格式输出 | 高 |
| ASR 识别 | Whisper 多语言语音转文字 | 高 |
| 音频分段 | 按时间戳切割音频 | 中 |

#### 3.3.3 视觉分析

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 图像描述 | BLIP-2 生成画面描述 | 高 |
| 物体检测 | YOLOv8 实时检测 | 中 |
| 场景分类 | 视频场景识别 | 中 |
| 情感检测 | 人物表情分析 | 低 |

### 3.4 多模态对齐模块

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 时间轴对齐 | 字幕、音频、画面同步 | 高 |
| 语义对齐 | 跨模态语义关联 | 高 |
| 手动校正 | 人工修正对齐结果 | 中 |

### 3.5 隐喻识别模块

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 概念隐喻 | 识别文本中的概念隐喻 | 高 |
| 视觉隐喻 | 识别画面中的隐喻表达 | 高 |
| 多模态隐喻 | 文本+画面协同隐喻 | 高 |
| 自定义规则 | 用户定义隐喻模式 | 中 |

### 3.6 不可译性识别模块

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 语言不可译 | 语音、词汇、句法层面 | 高 |
| 文化不可译 | 文化专有项、习语 | 高 |
| 语境不可译 | 语境依赖的表达 | 中 |

### 3.7 语料输出模块

| 功能 | 说明 | 优先级 |
|------|------|--------|
| JSON 导出 | 完整语料结构 | 高 |
| CSV 导出 | 统计分析格式 | 高 |
| 标注文件 | 质性分析格式 | 高 |
| 批量导出 | 多视频批量处理 | 中 |
| 统计报告 | 数据分布报告 | 中 |

## 五、部署架构

### 5.1 开发环境

使用 Docker Compose 快速启动：

```bash
docker-compose up -d
```

服务包括：
- PostgreSQL (端口 5432)
- Redis (端口 6379)
- MinIO (端口 9000/9001)

### 5.2 生产环境

推荐使用 Kubernetes 部署：

- 多副本 FastAPI 服务
- PostgreSQL 主从复制
- Redis Cluster
- MinIO 分布式存储

### 5.3 环境要求

| 组件 | 最低配置 | 推荐配置 |
|------|----------|----------|
| CPU | 4 核 | 8 核+ |
| 内存 | 8 GB | 16 GB+ |
| 磁盘 | 100 GB | 500 GB+ SSD |
| GPU | - | NVIDIA GPU (可选) |
